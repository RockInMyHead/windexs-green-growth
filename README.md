# Windexs Реклама - Сайт рекламного агентства

## О проекте

Этот проект представляет собой одностраничный сайт (SPA) для рекламного агентства Windexs, разработанный с использованием современных веб-технологий.

## Технологии

Проект построен с использованием следующих технологий:

- **Vite** - быстрый сборщик для разработки
- **TypeScript** - типизированный JavaScript
- **React** - библиотека для создания пользовательских интерфейсов
- **shadcn/ui** - компоненты пользовательского интерфейса
- **Tailwind CSS** - утилитарный CSS-фреймворк
- **Framer Motion** - анимации
- **React Router** - маршрутизация

## Локальный запуск

Для запуска проекта локально необходимо:

### Frontend

1. Установить зависимости:
```bash
npm install
```

2. Запустить development сервер:
```bash
npm run dev
```

Frontend будет доступен по адресу `https://wizary.windexs.ru`

### Backend (для отправки email)

1. Перейти в папку сервера:
```bash
cd server
```

2. Установить зависимости:
```bash
npm install
```

3. Настроить `.env` файл с API ключом от smtp.bz (смотрите server/README.md)

4. Запустить сервер:
```bash
npm start
```

Backend будет доступен по адресу `https://wizary.windexs.ru`

## Сборка проекта

Для сборки проекта в production:
```bash
npm run build
```

Проект использует современный стек технологий для быстрой и эффективной разработки. Все компоненты типизированы с помощью TypeScript, что обеспечивает высокую надежность кода.
